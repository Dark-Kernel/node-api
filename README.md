# Node Webscraper-API

### Simple script to get data from E-Commerce site:
    1. Amazon
    2. Flipkart
    3. Reliance Digital
    4. ShopClues


### Routes: 
    - /results?product=
    - /specs?link=
	- /email?id=<_emailid_>&link=<_track_link>
	- /cache
	- /suggestion?product=<name>
	- /wishlist?act=<i/d/r>&title=<>&link=<>&id=<email>
		* i :
				- All
		* r :
				- id
		* d : 	
				- link, id , title
